<h1>17_React Hook</h1>

<h2>1. React Hook</h2>

fitur baru di react 16.8. dengan hook kita dapat menggunakan state dan fitur react yang lain tanpa perlu menulis sebuah kelas.
Hooks dasar : 
- use state
- useContect
- Use Effect

<h2>2. Aturan hooks</h2>

- Hanya panggil hooks di tingkat atas (jangan manggil hooks dalam loops, conditions, nested function)
- Hanya Panggil hooks dari fungsi" react (memanggil dari komponen" fungsi react, memanggil custom Hooks)

<h2>3. Use Effect()</h2>

- memungkinkan kita melakukan efek samping (side effect) di dalam function component
- componentDidmount, componentdidUpdate dan componentWillUnMount
